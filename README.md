##  Group Assignment 2

### Question 1: Encryption & Decryption Program
This program reads a text file, encrypts it using custom shift rules, decrypts it back, and verifies that the original and decrypted files match.

### Question 2: Mathematical Expression Evaluator
This program reads mathematical expressions from a file, tokenises and parses them using recursive descent parsing, evaluates the results, and outputs them in the required format.
